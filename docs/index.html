安装说明
  • Windows：下载 .exe 后双击运行，按向导完成安装，支持自定义安装路径。
  • macOS：下载 .dmg 后打开，将应用拖入「应用程序」文件夹。首次运行如提示"无法验证开发者"，请在「系统设置 → 隐私与安全性」中点击「仍要打开」。
  • 应用需要连接内网数据库，请确保网络环境可访问服务器。
  • 有新版本时，应用会在登录后自动提示更新。
